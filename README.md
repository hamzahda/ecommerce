## Example Usage

**Usage with Docker**
Download or clone the project then
```
$ cd /Eshop
$ docker compose up
```
**Usage W/o Docker**
In this setup you'll need to open two <br>
Download or clone the project then run the following cmds in the first <br>
```
$ cd /Eshop/server
$ gradle bootRun
```
and the following in the second
```
$ cd /Eshop/application
$ npm run serve
```
